Bootstrap

A HTML, CSS and Javascript framework for developing responsive, mobile first projects from the web.

Originally built at Twitter, open-sourced to the world.

More about Bootstrap?

Why Use It?

  • Easy to get started, good documentation, huge community
  • Mobile-first (with 3.0) with easy-to-implement responsive design
  • Solid set of components and base elements
  • Javascript plugins allow for easy frontend interactions
  • A little designer zing when you don't have one

LESS (and SASS)

Pre-processing! Brings "real" coding to CSS. More maintainable, themable and extendable. Originally Ruby, now written in Javascript.

More about LESS?

It uses loops, can do calculations, it compiles.

SASS is a little more sophisticated, maybe better. Requires Ruby.

Either one is vastly better than just CSS.

Variables

Don't repeat yourself!


// Set the variable once
@uwPurple: #4b2e83;
// Then use throughout
h1, h2, h3 {
	color: @uwPurple;
}
.footer {
	background-color: @uwPurple;
}
// Get fancy
.light-background {
	background-color: lighten(@uwPurple, 20%);
}

// Compiles to
h1, h2, h3 {
	color: #4b2e83;
}
.footer {
	background-color: #4b2e83;
}
.light-background {
	background-color: #7a54c3;
}

Nesting

Stay sane. But beware too many descendants!


.main-content {
	padding: 1em 2em;
	.highlight {
		font-size: 20px;
		color: @uwPurple;
		&.alt {
			color: #fff;
			background-color: darken(@uwPurple, 10%);
		}
	}
}
// Compiles to
.main-content {
  padding: 1em 2em;
}
.main-content .highlight {
  font-size: 20px;
  color: #4b2e83;
}
.main-content .highlight.alt {
  color: #fff;
  background-color: #35215d;
}

// Lots of other LESS options to reverse order, etc.
h1 {
  color: red;
  .alt-container & {
    color: blue;  
  }
}
// Compiles to
h1 {
  color: red;
}
.alt-container h1 {
  color: blue;
}
